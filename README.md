# Trading Journal SaaS

A clean and powerful trading journal to track trades, analyze performance, and improve consistency.

🔗 **Live Demo:** https://trading-journal-saas.netlify.app/

## ✨ Features

- User authentication (Register / Login)
- Multiple trading accounts per user
- Add and view trades
- Detailed trade history with filtering
- View specific trade data
- Add and manage notes
- Dashboard with performance analytics and charts
- Data stored securely in database
- Contact section
- Logout functionality

## 🛠 Tech Stack

- Frontend: React, Context API, CSS Modules
- Backend: Node.js + Express.js
- Database: MongoDB
- Auth: JWT

## 🚧 Future Improvements

- Update / Delete trades
- Tagging + better filtering
- Export to CSV/PDF
- Social features (share trades, comments, likes)
- Strategy performance insights & smart analytics
